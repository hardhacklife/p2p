P2P File System

Interface professionnelle de gestion distribuée des fichiers : sélection du nœud, upload, téléchargement, création de fichiers texte et consultation rapide du stockage disponible.

Nœud actif : Node 5000

Choix du nœud

Sélectionne le nœud cible sur lequel tu veux exécuter les opérations.

Les requêtes seront envoyées vers le nœud actuellement sélectionné.

Upload de fichier

Ajoute un document existant dans le système distribué.

Téléchargement

Récupère un fichier en indiquant son nom exact.

Créer un fichier texte

Rédige et envoie directement un nouveau fichier texte sur le nœud choisi.
